Quick Help

FretFlow User Guide

1. Getting Started: The Basics

The first time you use FretFlow, your browser will ask for microphone permission. Please allow this, as the app needs to hear your instrument to give you feedback.

The core loop is simple:

  1. Choose a Workflow that matches your goal.
  2. Use the main action button for active practice workflows: Start Session, Start Study, Start Practice, or Start Run.
  3. In Library and Editor, use the workflow buttons and melody actions inside the panel instead of a top run button.
  4. Follow the current prompt or melody task.
  5. Play and use the live feedback to adjust your accuracy and timing.

2. Workflows, Options, and App Settings

  • Learn Notes: Use this workflow to train fretboard knowledge, note finding, intervals, scales, chords, and related note-study tasks.
  • Study Melody: Use this workflow for step-by-step melody study. The app waits for the correct note before moving forward.
  • Practice: Use this workflow to drill the melody continuously and build consistency before a final performance run.
  • Perform: Use this workflow for the final full-run check. The melody keeps moving and your timing and pitch are graded in real time.
  • Library: Use this workflow to browse, preview, export, and organize melodies without changing source notes.
  • Editor: Use this workflow to create new melodies, import TAB/MIDI/MuseScore/Guitar Pro, and edit source notes in the preview or timeline.
  • Workflow panels: Each workflow now shows its relevant panels directly, without extra setup buttons. Learn Notes shows note practice controls, melody workflows show melody controls, and Perform adds only the run-specific settings.
  • App Settings: Open the gear icon for global preferences like Instrument, Tuning, note naming (Sharps/Flats), input devices, and rhythm timing window.
  • Difficulty:
    • Natural Notes Only: Restricts challenges to A, B, C, D, E, F, G. Perfect for beginners.
    • All Notes: Includes sharps/flats for a complete fretboard workout.
  • Mode: In Advanced view, detailed sub-modes still live inside the note-training workflow.
  • Import and melody editing: These are most useful inside Editor, while Library is better for preview, export, and cleanup.

3. The Session Area: Where the Action Happens

  • Prompt & Play Sound Button: The big text is your challenge. The speaker icon plays a high-quality sound of the target note.
  • Main Action & Status: The top button changes with the active workflow. The status badge shows the current state, such as Ready or Listening.
  • Live Tuner: Perfect your intonation. The needle shows if you're sharp (♯), flat (♭), or in-tune (green). The cents display gives you ultra-precise feedback.
  • Volume Indicator: Shows if the microphone is picking up your instrument.
