FretFlow User Guide

1. Getting Started: The Basics

The first time you use FretFlow, your browser will ask for microphone permission. Please allow this, as the app needs to hear your instrument to give you feedback.

The core loop is simple:

  1. Click "Start Session".
  2. Look at the prompt (e.g., "Find: G on D string").
  3. Play that note on your instrument.
  4. Get instant feedback on your accuracy!

2. Practice Setup, Session Settings, and App Settings

  • Practice Setup: Use the Practice Setup button to choose what you are training, such as mode, difficulty, and core mode-specific material. It auto-collapses when a session starts.
  • Session Settings: Use Session Settings for session planning and visual aids, such as curriculum, goal, fret range, Show All Notes, and Show String Toggles. In some modes (for example Rhythm), this panel is hidden to reduce clutter.
  • App Settings: Open the gear icon for global preferences like Instrument, Tuning, note naming (Sharps/Flats), input devices, and rhythm timing window.
  • Difficulty:
    • Natural Notes Only: Restricts challenges to A, B, C, D, E, F, G. Perfect for beginners.
    • All Notes: Includes sharps/flats for a complete fretboard workout.
  • Mode: This lives in Practice Setup.
  • Curriculum / Goal / Pace / Frets: These live in Session Settings.

3. The Session Area: Where the Action Happens

  • Prompt & Play Sound Button: The big text is your challenge. The speaker icon plays a high-quality sound of the target note.
  • Start/Stop & Status: Start a session with the main button. The status badge in the top-right shows the current state (Ready / Listening / etc.).
  • Live Tuner: Perfect your intonation. The needle shows if you're sharp (♯), flat (♭), or in-tune (green). The cents display gives you ultra-precise feedback.
  • Volume Indicator: Shows if the microphone is picking up your instrument.
